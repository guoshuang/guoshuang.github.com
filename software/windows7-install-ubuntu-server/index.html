恢复 grub

hp 安装 ubuntu 以后没有 grub 菜单

  1. 找到你安装ubuntu的安装光盘或u盘
  2. 进入liveCD模式，启动后打开终端(ctrl + alt +T)
  3. sudo -i ，进入根用户下。
  4. fdisk -l ，查看你的ubuntu系统安装位置，我的是sda3.
  5. mount /dev/sda3 /mnt ，挂载上硬盘。
  6. grub-install —root-directory=/mnt /dev/sda，把启动项安装到硬盘启动信息中（注意/mnt和/dev/sda有个空格）
  7. 重启电脑
  8. sudo update-grub2 ，更新启动列表，引导其它系统。
    成功解决，这个很蛋疼的问题，其它品牌电脑没有这样的问题。其实上述步骤也解决
    grub损坏的问题。

添加用户

useradd 和 adduser 区别是

useradd 是系统自带。adduser 是前者的 perl 脚本，后者更好用！

显示所有组

cat /etc/group

但前用户所在组

groups

加用户 guoshuang（带home目录）

sudo adduser guoshuang

把 guoshuang 加到 这些组中

sudo usermod -G adm,cdrom,sudo,dip,plugdev,sambashare,lpadmin -a guoshuang

注意：ssh 连接的话，还需要退出重新连接一次！！！

-G, —groups GROUP1[,GROUP2,…[,GROUPN]]]

A list of supplementary groups which the user is also a member of.
Each group is separated from the next by a comma, with no
intervening whitespace. The groups are subject to the same
restrictions as the group given with the -g option.

If the user is currently a member of a group which is not listed,
the user will be removed from the group. This behaviour can be
changed via the -a option, which appends the user to the current
supplementary group list.
