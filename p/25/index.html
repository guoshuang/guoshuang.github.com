simulate google ajax indacator 2

最后更新: Wed Sep 10 2014 12:29:11 GMT+0800

依稀记得还有一种 google 的 ajax 效果






原理

内含4个div，每个里面2个半圆。依次其中半个 animation 3D 旋转。大小、速度、颜色可定义。有两个细节：

  1. 整个背景是第4个颜色，否则（红色后面露出白色）就漏了。
  2. “毛边”的问题。因为颜色叠加，所以有“毛边”。所以在 :after 放一个白色边框，压一下。

查看源文件：less css

@ball:30px; //ball size
@color:'red,green,blue,#f90'; //颜色数组
@speed:3s;

function checkInput(type) {
    var input = document.createElement("input");
    input.setAttribute("type", type);
    return input.type == type;
}

jQuery.fn.selectText = function(){
   var doc = document;
   var element = this[0];
   console.log(this, element);
   if (doc.body.createTextRange) {
       var range = document.body.createTextRange();
       range.moveToElementText(element);
       range.select();
   } else if (window.getSelection) {
       var selection = window.getSelection();        
       var range = document.createRange();
       range.selectNodeContents(element);
       selection.removeAllRanges();
       selection.addRange(range);
   }
};
