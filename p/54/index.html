flatland theme 4 vim

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

下载 vim 的主题文件，拷贝到 ~/.vim/colors/ 目录中。

cp flatland.vim ~/.vim/colors/flatland.vim

然后打开 vim，执行

:colorscheme flatland

或者

:colo flatland

但下次打开 vim 还是默认主题，需要建立 ~/.vimrc

vi ~/.vimrc

colo flatland

sublime 使用技巧

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

打开刚关掉的文件

+shift+t

自动绕行（word wrap） 快捷键

选择 Preferences - Key Bindings-User

{ "keys": ["Command+alt+w"], "command": "toggle_setting", "args": {"setting": "word_wrap"} }

Side Bar 高亮当前文档（类似 eclipse 的 link）

右键 + Reveal in Side Bar

使用project（Side Bar 显示常用目录）

使用 project 可以迅速定位那些常用的文件们。sidebar 的那些目录会被保存。拖动文件夹到 sidebar 中；删除用 右键点击 根目录 Remove Folder from Project。Projrct - Save Project as

批量改名（Multiple cursors）

+B 选择和当前选择一样的内容（下一个）
+Ctrl+G 选择和当前选择一样的内容（全部）

直接写，所有地方一起修改

开关 sidebar

默认是 +k 然后 +b


而在我的mac +k 是删除当前行 ）直接改成 +b，选择 Preferences - Key Bindings-User

{ "keys": ["Command+b"], "command": "toggle_side_bar"}

注意：+b 默认是 { “keys”: [“super+b”], “command”: “build” }

文件查找和搜索

  • +p 搜索 project 中的文件名
  • 输入 #123,在当前文件所有 123 的内容！等同于 +f 或者 +I 搜索
  • 输入 :12 切换到第 12 行

快速执行命令

+Shift+p 搜索相关命令。

查找类或者函数

+r 比如打开一个 .js 文件，会显示所有函数和类的名称

多行（列）布局

View - Layout - Columns:2 等等方便对比文件

注意：Command 或者 Cmd 或者 super 实际上都是指苹果的“花”键 ⌘ html entity 代码是 &amp;#8984;

书写 html 标签（前后标签一起修改）

  • 选择一段文字，ctrl+shift+w
  • 直接 ctrl+shift+w
  • 双击开始 <tag> ，然后 +d 前后标签一起修改

列选择

ctrl+shift+↑ 或者 ctrl+shift+↓

我的Mac 下的 ctrl+shift+上下键 被 mission control 占领了，只好修改 Preferences - Key Bindings-User

{ "keys": ["super+alt+up"], "command": "select_lines", "args": {"forward": false} },
{ "keys": ["super+alt+down"], "command": "select_lines", "args": {"forward": true} }

官方说 Left Mouse Button+Option 也可以；但我的 mac 是 +Left Mouse Button

注释和取消

+/

+alt+/

合并下一行

+j

多重选择

+鼠标拖选 或者 双击

Mac Finder 显示隐藏文件

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

Finder 显示隐藏文件（设置并重启 Finder）

defaults write com.apple.Finder AppleShowAllFiles YES;killall -HUP Finder

关掉用

defaults write com.apple.Finder AppleShowAllFiles NO;killall -HUP Finder

sublime2 主题 flatland

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

看到 lucifr 推荐的 flatland for sublime theme，很漂亮，还有图标。

  1. 下载 Flatland 主题。

  2. 解压缩后将其中的 “Theme - Flatland” 文件夹拷贝到 Sublime Text 2 的 Packages 目录中，你可以通过 Sublime Text 2 的菜单 “Preferences > Browse Packages …” 打开这个目录。

  3. 修改 Preferences 文件激活主题，通过 Sublime Text 2 的菜单 “Preferences > Settings - User” 可打开用户配置文件，在其中添加（或修改原来的设置）：

{
"theme": "Flatland.sublime-theme",
"color_scheme": "Packages/Theme - Flatland/Flatland.tmTheme"
}

Mac 换图标

Finder 打开 Applications，找到 sublime 右键，Show Package Contents，找到 Resources 目录，里面都是 icns，替换同名文件即可

jquery prop

最后更新: Wed Sep 10 2014 12:37:50 GMT+0800

jquery prop() (property) 获得属性。和 attr() 很类似，但区别也不少。推荐优先使用 prop()，有些 非显式声明 的属性只能用 prop。（jquery 1.6 以前的 attr 不能获得 实时 的属性）

prop 不能获得自定义属性值（jquery 1.9.1） attr(‘data-url’) 可以
