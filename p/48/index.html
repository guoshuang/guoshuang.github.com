express+jade+stylus+coffee

最后更新: Wed Sep 10 2014 12:36:06 GMT+0800

前言

Express 是 Node 的一个流行框架之一，npm 是 Node 的包管理器。jade 是 html 模版引擎，stylus 是 css 预处理器，和 LESS 或者 SASS 差不错。coffee 用来写 javascript。Nib 是 stylus 一个工具，用来加入 vendor-prefix，-webkit-transform(chrome,safari) -moz-transform(mozilla,firefox) -o-transform(opera)

白话版：express 就是 node，npm 是 node 安装东西的工具，jade 用来写 html5，coffee 写javascript，stylus 写 css，nib 用来帮助生成 这些混乱的浏览器厂家 css 标记。总之，一切为了 － 快！

hexo zero file

最后更新: Wed Sep 10 2014 12:37:50 GMT+0800

hexo 经常出错，生成 0 字节大小的文件。而且 hexo generate 也不能修复，必须手工删除，下次 hexo generate 才行

进入 hexo 的 public 目录

找到所有 0 字节文件

find . -size 0

删除这些 0 字节文件 看清楚！别弄错了目录！

find . -size 0 -exec rm {} +

man pagedown pageup

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

比如 man find 内容很长，找东西很不方便。

在 man 中

Mac 没有pageup down键，所以只能用 b(back) f(forward)

f 或者 z 向前（下）翻页

b 或者 w 向后（上）翻页

q 退出

/AAA

向前（下）查找AAA

?AAA

向后（上）查找AAA

n 继续查找 N 反方向继续查找

d 向前（下）半页
u 向后（上）半页

h 快捷键 帮助

在 vim 中

ctrl+f 向前（下）翻页

ctrl+b 向后（上）翻页

其它办法

man find > 1.txt 然后打开 1.txt 慢慢看

man find | pbcopy 已经拷贝到系统剪贴板了，找个文本编辑器粘贴吧
