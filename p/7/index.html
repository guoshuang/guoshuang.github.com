var_dump 和 print_r 区别

The var_dump function displays structured information about variables/expressions including its type and value. Arrays are explored recursively with values indented to show structure. It also shows which array values and object properties are references.

The print_r() displays information about a variable in a way that’s readable by humans. array values will be presented in a format that shows keys and elements. Similar notation is used for objects.

Example:

$obj = (object) array(‘qualitypoint’, ‘technologies’, ‘India’);
var_dump($obj) will display below output in the screen.

object(stdClass)#1 (3) {
[0]=> string(12) “qualitypoint”
[1]=> string(12) “technologies”
[2]=> string(5) “India”
}
And, print_r($obj) will display below output in the screen.

stdClass Object (
[0] => qualitypoint
[1] => technologies
[2] => India
)

解析域名到花生壳

最后更新: Wed Sep 10 2014 12:39:11 GMT+0800

貌似 VPS 都蛮贵的，MT,godaddy 等都是 $30/月，穷人们只好另辟它迳。

路由器上设置花生壳

在 虚拟服务器 或者 端口映射 那里打开 80 到内网 192.168.1.x 80。(ssh 端口 22)

设置花生壳得到 xxxxx.eicp.net 地址。测试看是否可用。

设置 dns cname 到 花生壳地址

我的域名服务商是 namecheap。（鉴于国内 域名、主机服务，罪犯一样的体验，我的域名和空间都逃了出去）

  • Click My Account
  • Click Manage Domains.
  • Under Domain Names, click the domain you wish to modify
  • Click All Host Records in the menu on the left
  • Under Sub Domain Settings type the subdomain name you chose for your website in LaunchRock.
  • For example, if you chose signup.yoursite.com, enter: signup.
  • In the IP Address / URL field enter host.launchrock.com
  • In the Record Type menu select CNAME (Alias)
  • Click Save Changes

比如我的域名是 guoshuang.com 而我需要 cname lab.guoshuang.com 的话：

  • 第一个填（Sub Domain） lab
  • 第二个填（IP Address / URL） xxxx.eicp.net.
  • ttl 无所谓

如何测试

namecheap 的 dns 服务器是

  • freedns1.registrar-servers.com
  • freedns2.registrar-servers.com
  • freedns3.registrar-servers.com
  • freedns4.registrar-servers.com
  • freedns5.registrar-servers.com

打开终端执行

nslookup

>server freedns1.registrar-servers.com 使用哪个dns服务

>lab.guoshuang.com 测试

这确结果应该类似：

lab.guoshuang.com canonical name = xxxxxxx.eicp.net.

canonical name(CNAME) 规范名

测试正确的话，过几小时或几天的话，这个 dns 就会生效。

恢复 grub

hp 安装 ubuntu 以后没有 grub 菜单

  1. 找到你安装ubuntu的安装光盘或u盘
  2. 进入liveCD模式，启动后打开终端(ctrl + alt +T)
  3. sudo -i ，进入根用户下。
  4. fdisk -l ，查看你的ubuntu系统安装位置，我的是sda3.
  5. mount /dev/sda3 /mnt ，挂载上硬盘。
  6. grub-install —root-directory=/mnt /dev/sda，把启动项安装到硬盘启动信息中（注意/mnt和/dev/sda有个空格）
  7. 重启电脑
  8. sudo update-grub2 ，更新启动列表，引导其它系统。
    成功解决，这个很蛋疼的问题，其它品牌电脑没有这样的问题。其实上述步骤也解决
    grub损坏的问题。
