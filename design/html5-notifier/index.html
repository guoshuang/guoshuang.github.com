声音文件

念中文需要在 Preferences - Speech - Text to Speech ，system Voice 中选择 customize,然后选中文，需要安装600M的语言包

say 老郭牛逼 -o 1.aiff

听听对不

afplay 1.aiff

下载编译 lame（或者 直接下载 lame mac），然后转换 mp3。当然，用某些播放器也可以转换。

lame 1.aiff 1.mp3

测试下

afplay 1.mp3

编译 lame

和 *nix 世界一样。解压，终端进入文件夹

./configure 检查环境

make 编译

sudo make install 编译安装
